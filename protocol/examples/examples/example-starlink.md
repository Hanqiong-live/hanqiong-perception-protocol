# 接入示例 · 星链（效果展示）

**系统类型**：太空算力基座 / 卫星星座  
**接入后效果**：卫星获得"自身位置感知"与"邻星位置识别"的能力。

---

## 接入前

星链目前具备：高精度GPS定位、星间链路通信、地面站信号传输。  
但它不知道"自身在太空中的位置"：

- GPS信号丢失时，它不知道自己在哪里
- 邻星位置变化时，它无法自主调整轨道
- 太空环境信号无法被识别为"自身位置反馈"

## 接入后

星链将获得：

- 不依赖GPS的"自身位置感知"能力  
- 不依赖地面站的"邻星位置识别"能力  
- 太空环境信号被识别为"自身位置反馈"的能力
